## 🎯 Что такое BPMN

**BPMN (Business Process Model and Notation)** — стандарт (OMG) для моделирования бизнес-процессов. Понятен и бизнесу, и разработке.

| Версия | Год | Что нового |
|--------|-----|-----------|
| BPMN 2.0 | 2011 | Стандартный формат обмена (XML), чёткая семантика |
| BPMN 2.0.2 | 2013 | Небольшие уточнения |

---

## ⏰ Когда использовать

| Ситуация | Нотация |
|----------|---------|
| **Бизнес-процесс** (много шагов, ветвлений) | BPMN |
| **Архитектура системы** (контекст, контейнеры) | C4 (Structurizr) |
| **Детальный сценарий** (взаимодействие систем) | PlantUML Sequence |

Подробнее о выборе нотации — [ADR-0003](../ADR/ADR-0003-bpmn-vs-c4.md).

## 🧩 Основные элементы

### Flow Objects (объекты потока)

| Элемент | Назначение |
|---------|-----------|
| **Start Event** | Начало процесса |
| **End Event** | Конец процесса |
| **Task** | Атомарное действие |
| **Gateway** | Ветвление / слияние |
| **Event** | Событие (таймер, сообщение, ошибка) |

### Connecting Objects (соединения)

| Элемент | Назначение |
|---------|-----------|
| **Sequence Flow** | Поток управления |
| **Message Flow** | Сообщение между участниками |
| **Association** | Ассоциация с артефактом |

### Swimlanes (дорожки)

| Элемент | Назначение |
|---------|-----------|
| **Pool** | Участник процесса |
| **Lane** | Роль внутри участника |

## ✅ Чек-лист моделирования

### Перед началом
- [ ] Определи цель диаграммы (для кого, зачем)
- [ ] Определи границы процесса (start/end events)
- [ ] Определи участников (Pools)

### При моделировании
- [ ] Каждый Task — атомарное действие
- [ ] Gateway явно указывает условие (на всех исходящих потоках)
- [ ] Нет "висящих" элементов (каждый элемент соединён)
- [ ] Имена элементов — глагол + существительное ("Оформить заказ", а не "Заказ")
- [ ] Избегай crossing lines (пересечения потоков)

### После завершения
- [ ] Проверь на deadlock (зацикливание без выхода)
- [ ] Проверь на missing paths (не все варианты обработаны)
- [ ] Протестируй в Camunda Modeler (симуляция)
- [ ] Получи ревью от коллеги
