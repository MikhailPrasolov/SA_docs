# Тестирование моделей транскрипции

Скрипт для сравнения разных моделей распознавания речи на одном аудио/видеофайле.
Позволяет выбрать лучшую модель для русского языка по качеству и скорости.

---

## Расположение

```
scripts/
├── transcribe_test.ps1    # Основной скрипт
└── transcribe_test.bat    # Обёртка для запуска из cmd/проводника
```

---

## Быстрый запуск

```powershell
cd scripts

# Сравнить все модели на одном файле
.\transcribe_test.ps1 "C:\video\test.mp4"

# Использовать только конкретную модель (без сравнения)
.\transcribe_test.ps1 "C:\audio\speech.wav" -Model large-v3-turbo
```

---

## Параметры

| Параметр | По умолчанию | Описание |
|----------|-------------|----------|
| `InputPath` | — | Путь к файлу (обязательный, любой формат FFmpeg) |
| `-Model` | `all` | Какая модель: `large-v3`, `large-v3-turbo`, `medium`, `small`, `all` |
| `-Language` | `ru` | Язык (код языка или `auto`) |
| `-OutputDir` | рядом с файлом | Куда сохранить результаты |

---

## Доступные модели

| Параметр | Модель | Размер | Качество | Скорость |
|----------|--------|:------:|:--------:|:--------:|
| `large-v3` | Systran/faster-whisper-large-v3 | ~3 GB | ⭐⭐⭐⭐⭐ | 🐢 |
| `large-v3-turbo` | Systran/faster-whisper-large-v3-turbo | ~1.6 GB | ⭐⭐⭐⭐ | ⚡ Быстрая |
| `medium` | Systran/faster-whisper-medium | ~1.5 GB | ⭐⭐⭐ | ⚡⚡ Быстрая |
| `small` | Systran/faster-whisper-small | ~500 MB | ⭐⭐ | ⚡⚡⚡ Оч. быстрая |

## Формат вывода

Транскрипция разбивается на **абзацы** — каждая новая мысль с новой строки:

```
Сегодня мы тестируем разные модели распознавания речи.

Первая модель large-v3 показывает отличное качество, но работает медленно.

Вторая модель large-v3-turbo почти не уступает в качестве, но в два раза быстрее.
```

Принцип разбивки:
- **Пауза < 0.3 с** — продолжение того же предложения
- **Пауза 0.3–1.0 с** — новое предложение
- **Пауза > 1.0 с** — новый абзац

## Пример сравнения

```powershell
.\transcribe_test.ps1 "D:\лекция.mp4"
```

Результат:
```
==========================================
  RESULTS
==========================================
  large-v3       → 12 paragraphs | 04:23 | RTF: 1.25
  large-v3-turbo → 12 paragraphs | 01:35 | RTF: 0.45
  medium         → 11 paragraphs | 00:52 | RTF: 0.25
  small          → 9 paragraphs  | 00:28 | RTF: 0.13
```

**RTF (Real-Time Factor)** — коэффициент реального времени:
- `1.00` = 1 минута аудио обрабатывается 1 минуту
- `0.50` = 1 минута аудио обрабатывается 30 секунд

**Что смотреть:**
1. **Качество** — открыть `.txt` файлы и сравнить
2. **Количество абзацев** — если модель пропустила часть текста, абзацев будет меньше
3. **RTF** — чем ниже, тем быстрее модель

Файлы сохраняются рядом с исходным:
```
lecture.mp4 → lecture__large-v3.txt
              lecture__large-v3-turbo.txt
              lecture__medium.txt
              lecture__small.txt
```

---

## Рекомендации по выбору

| Если нужно | Ставьте |
|------------|---------|
| **Максимальное качество**, время не важно | `large-v3` |
| **Хорошее качество + приемлемая скорость** | `large-v3-turbo` |
| **Быстро и достаточно качественно** | `medium` |
| **Черновик / быстрый просмотр** | `small` |

**Для русского языка** оптимальный баланс: **large-v3-turbo**. На тестах он почти не уступает large-v3 по качеству, но работает в 2-3 раза быстрее.
