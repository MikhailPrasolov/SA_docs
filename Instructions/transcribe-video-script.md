# Транскрипция аудио и видео через PowerShell + faster-whisper

PowerShell-скрипт для транскрипции аудио/видеофайлов в текст через `faster-whisper`. Работает на CPU (CUDA не требуется).

---

## Расположение

```
scripts/
├── transcribe_video.ps1    # Основной скрипт
└── transcribe_video.bat    # Обёртка для запуска из cmd/проводника
```

---

## Поддерживаемые форматы (входные)

Любые форматы, которые читает FFmpeg:

- **Видео:** MP4, MKV, AVI, MOV, WebM, FLV, WMV, MPEG, TS
- **Аудио:** MP3, WAV, FLAC, M4A (AAC), OGG (Vorbis/Opus), WMA, AMR, AIFF

Внутри FFmpeg конвертирует аудиодорожку в WAV (16 kHz, 16-bit, mono) — временный файл удаляется после транскрипции.

---

## Быстрый запуск

### Из PowerShell

```powershell
cd scripts
.\transcribe_video.ps1 "C:\video\sample.mp4"
```

Результат (`sample.txt`) появится рядом с видео.

### Из cmd (командная строка)

```cmd
cd scripts
.\transcribe_video.bat "C:\video\sample.mp4"
```

> В PowerShell обязательно писать `.\` перед именем `.bat` файла.

---

## Параметры

```powershell
.\transcribe_video.ps1 <InputPath> [OutputPath] [-ModelSize <str>] [-Language <str>] [-ForceCPU]
```

| Параметр | По умолчанию | Описание |
|----------|-------------|----------|
| `InputPath` | — | Путь к файлу (обязательный, любой формат FFmpeg) |
| `OutputPath` | рядом с входным `.txt` | Куда сохранить результат |
| `-ModelSize` | `large-v3` | Размер модели: `tiny`, `base`, `small`, `medium`, `large-v3` |
| `-Language` | `ru` | Язык транскрипции (код языка, или `auto`) |
| `-ForceCPU` | нет | Принудительно CPU |

### Примеры

```powershell
# Аудиофайл
.\transcribe_video.ps1 "C:\audio\запись.mp3"

# Видео, быстрая транскрипция (модель medium)
.\transcribe_video.ps1 "D:\лекция.mp4" -ModelSize medium

# Английский язык
.\transcribe_video.ps1 "C:\video.mp4" -Language en

# Сохранить в конкретный файл
.\transcribe_video.ps1 "лекция.m4a" "D:\result.txt"
```

## Как это работает

1. FFmpeg извлекает/конвертирует аудио в WAV (16 kHz, 16-bit, mono)
2. `faster-whisper` загружает модель и транскрибирует (CPU, compute_type=`int8`)
3. Результат сохраняется в `.txt` (по одной строке на сегмент)
4. Временный WAV удаляется

## Модели

| Модель | Размер | Качество | Скорость на CPU |
|--------|--------|----------|-----------------|
| `tiny` | 75 MB | Плохо | ⚡ Очень быстро |
| `base` | 150 MB | Средне | ⚡ Быстро |
| `small` | 500 MB | Хорошо | ⚡ Быстро |
| `medium` | 1.5 GB | Очень хорошо | 🐢 Средне |
| `large-v3` | 3 GB | Отлично | 🐢 ~1-2 мин на 10 сек аудио |

Рекомендуется `large-v3` для русского языка. Для быстрой транскрипции — `medium`.

## Возможные проблемы

**"Файл не найден" даже если файл существует**
→ В пути есть невидимый символ. Набрать путь вручную, без копирования из проводника.

**"ffmpeg не найден"**
→ Установить: `winget install "FFmpeg (Essentials Build)"`

**"faster_whisper не установлен"**
→ `py -m pip install --user faster-whisper`

**Долгая транскрипция (40 мин видео — до 80 мин)**
→ Использовать `-ModelSize medium` — будет в 3-5 раз быстрее.

**Кракозябры вместо текста**
→ Проверить параметр `-Language`. Для русского — `ru`, или `auto`.
